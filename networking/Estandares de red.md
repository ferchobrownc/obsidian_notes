
las redes usan protocolos,  
- ethernet: rije tarjeta de interfaz de red 
- ip: rige la comunicacion de fuente original hasta su destino final
- tcp: protofocolo de control de transmision, ganarantiza que la informacion llegue alli, y si no llega envie un mensaje
- http: protocolo de transferencia de hipertexto, 

pila de protocolos, tcp/ip model
- network acceso: controla los disposisitivos del hardware y los medios que forman la red
- internet: determina el mejor camino a traves de una red
- transporte: admite la comunicacion entre disstintos dispositivos a traves de diversas redes
- aplicacion: representa datos para el usuario mas el control de conficicacion y de dialogo 




| Capa del modelo OSI | Descripción                                                                                                                                                                                                                          |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 7. Aplicación       | La capa de aplicación contiene protocolos utilizados para comunicaciones proceso a proceso.                                                                                                                                          |
| 6. Presentación     | La capa de presentación proporciona una representación común de los datos transferidos entre los servicios de la capa de aplicación                                                                                                  |
| 5. Sesión           | La capa de sesión proporciona servicios a la capa de presentación para organizar su diálogo y administrar el intercambio de datos.                                                                                                   |
| 4. Transporte       | La capa de transporte define los servicios para segmentar, transferir y re ensamblar los datos para las comunicaciones individuales entre terminales                                                                                 |
| 3. Red              | La capa de red propociona servicios para intercambiar los datos individuales en la red entre terminales identificados                                                                                                                |
| 2. Enlace de Datos  | Los protocolos de la capa de enlace de datos describen los metodos para intercambiar tramas de datos entre dispositivos en un medio común.                                                                                           |
| 1.  Física          | Los protocoles de la capa física describen los medios mecánicos, eléctricos, funcionales y de procedimiento para activar, mantener y desactivar conexiones físicas para una transmisión de bits hacia y desde un dispositivo de red. |
