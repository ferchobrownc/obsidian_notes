## 🚀 Qué representa este proyecto

Este repositorio forma parte de mi proceso de aprendizaje en ciberseguridad.

Aquí documento prácticas reales con laboratorios, comandos Linux, Bash y conceptos de redes, con el objetivo de desarrollar habilidades técnicas sólidas mediante la práctica constante.

No es un repositorio de teoría, sino de aprendizaje activo.

## 🧪 Qué practico aquí

- Retos de OverTheWire (Bandit)
- Uso de Linux en terminal
- Bash y manejo de archivos
- Laboratorios con Docker
- Fundamentos de redes
- Resolución de problemas en línea de comandos
